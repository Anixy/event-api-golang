# event-api-golang
An backend API for event application

# GET STARTED

Here is the steps to run it with `docker-compose`

```bash
#move to directory
$ cd workspace

# Clone into YOUR $GOPATH/src
$ git clone https://github.com/Anixy/event-api-golang.git

#move to project
$ cd event-api-golang

# Build the docker image first
$ go mod download

# Run the application
$ go run .

# Execute the call
$ curl localhost:8080
```
